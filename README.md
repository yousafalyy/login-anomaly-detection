# Login Anomaly Detection System 🔐

## 📌 Project Overview
This project detects unusual login behavior using Machine Learning.

## 🚀 Features
- Detects abnormal login times
- Identifies suspicious locations
- Tracks device-based anomalies
- Uses Isolation Forest algorithm

## 🧠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 🤖 Model Used
Isolation Forest Algorithm

## 📊 Output
Classifies login attempts as:
- Normal
- Anomaly

## 📁 File
- `login_anomaly.ipynb` → Main project notebook
